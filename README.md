# Proceso de Submission para Monad Blitz CDMX

1. Visita el repositorio `blitz-CDMX` ([link aquí](https://github.com/fruteroclub/blitzi-CDMX)) y haz un fork.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376176/blitz_1_jkmwqq.png)

2. Ponle el nombre de tu proyecto, una descripción corta, asegúrate de estar haciendo fork de la rama `main` y haz clic en `Create Fork`.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376177/blitz_2_awkuce.png)

3. En tu fork puedes hacer todos los cambios que quieras: agregar el código de tu proyecto, crear ramas, agregar información al `README.md`, puedes cambiar todo lo que necesites.

4. Una vez que hayas terminado tu proyecto y estés listo para enviarlo, crea un pull request.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376179/blitz_3_kfd3ua.png)
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376180/blitz_4_le1wnj.png)

5. Asegúrate de crear el pull request hacia el repositorio correcto: `fruteroclub/blitz-CDMX`.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376180/blitz_5_thrvsf.png)

6. Verifica que aparezca "Able to merge" (Listo para fusionar). Cuando lo veas, puedes hacer clic en `Create Pull Request`.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376175/blitz_6_rp87zq.png)

7. Dale a tu pull request el nombre de tu proyecto y una descripción detallada (describe todo lo que puedas sobre tu proyecto, incluso puedes agregar links a videos demo). Luego haz clic en `Create pull request`.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376176/blitz_7_bgbzef.png)

8. Finalmente, verifica que creaste tu pull request correctamente revisando el repositorio donde se creó y las ramas de origen y destino del pull request.
![image](https://res.cloudinary.com/ddejtxqjq/image/upload/v1769376176/blitz_8_tnjkqo.png)
